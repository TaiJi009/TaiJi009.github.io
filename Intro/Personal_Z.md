# 个人网站构建（思维导图整理）

## 一、示例：张滨的个人网站

### 1. 整体信息

- **类型**：个人主页
- **目的**：展示作品 / 联系我

### 2. 页面结构

#### 首页

- **Logo**：大写字母 **Z**
- **导航栏**：页面**右上角**
- **主视觉文案**：`HI HERE IS BEBSON`
- **操作**：主区域下方放置 **Contact Me** 按钮

#### 关于我页

- 内容：**自由发挥**

#### 联系我页

- 含 **头像**
- **联系方式**：微信、GitHub、小红书账号

#### 作品页

- 内容：**自由发挥**

### 3. 导航与跳转

| 项目 | 规格 |
|------|------|
| 导航栏位置 | 页面**右上角固定** |
| 导航项 | 首页、关于我、联系我、作品 |
| 跳转方式 | 点击跳转至对应页面 |

### 4. 视觉风格

在张滨个人作品集语境下，延续**莫兰迪系低饱和配色**：首页主视觉文案与 Logo「Z」置于沉静背景之上，底可选用灰调米色或浅灰蓝，文字用炭灰或灰褐色；导航与「Contact Me」建议从陶土、灰粉、雾霾蓝等色相中择一作为全站**唯一强调色**，悬停时只做明度或饱和度的细微抬升，不另引入高饱和跳色。

各页面（关于我、作品、联系）保持同一套灰阶与色相阶梯，头像与社交入口图标可套一层极浅的统一 tint，使微信、GitHub、小红书在视觉上「同属一个陈列柜」。作品页以网格或瀑布形式展示时，卡片间距与圆角全站统一；若有页面切换或内容淡入，**宜短时、曲线柔和**（交互动效保持拉姆斯式克制），气质沉静但不拖沓。**整体创意关键词**：画廊感、柔光、少色、材质轻、作品优先。

### 5. 配色方案

在「张滨」站点上落实莫兰迪体系时，遵循莫兰迪分层配色的一般逻辑；以下为**偏向沉静蓝灰 + 暖灰**的一组示例落位（实现时可再微调节以满足对比度无障碍）。

| 类型 | 配色 |
|------|------|
| **主色（背景色）** | **页底**：暖灰白或灰调燕麦；**首页 hero 区**：浅灰蓝灰（雾霾向）；**内容卡 / 作品栅格底**：比页底略亮半阶的冷灰白，统一「展览墙面」感。 |
| **强调色** | 全站只选 **一种**：推荐 **雾霾蓝（带灰）** 或 **陶土赭（带灰）**，用于导航当前态、「Contact Me」、外链与表单主按钮；禁用荧光绿、纯黑描边按钮。 |
| **文字色** | 主标题与 Logo 旁文案：**炭灰～灰褐**；**内页正文**：再略浅的灰褐或灰蓝黑；**社交与说明小字**：降对比度的灰阶。若需标题层次，使用 **灰褐 → 深灰褐** 的极弱渐变，与上述视觉风格中的字体层次处理一致。 |
| **辅助色** | 头像描边、图标衬底、分隔线：**灰蓝一阶或灰粉一阶**（莫兰迪 secondary）；作品缩略图外框用 **极浅 neutral 线色或低透明阴影色**，与高饱和描边区分。 |

### 6. 交互效果

交互上采用 **拉姆斯式克制**：本站在莫兰迪配色与画廊感前提下，交互以「少、准、统一」为纲；动效须按拉姆斯「好设计十原则」逐条自问 **是否可删、是否诚实、是否碍读**。

| 效果类型 | 规格（结合本站结构） |
|----------|----------------------|
| **页面切换** | 首页 / 关于我 / 联系我 / 作品之间：**统一短淡入淡出**（建议 200～280 ms，单一路线缓动）。无整页翻转、无全屏过渡遮罩叙事。**原则对照**：不显眼的、尽可能少的。 |
| **滚动** | **无滚动驱动装饰**（无不随内容的 stagger 渐入队列）。正文与作品网格随系统滚动即可。**原则对照**：实用的、环境负担少的。 |
| **导航与主操作** | 右上固定导航：当前项用 **强调色 + 字重或细下划线** 静态区分；hover 仅 **明度一阶或 opacity**，与《配色方案》一致。「Contact Me」 hover/active 同上，**禁止**大幅 scale 或强阴影。**原则对照**：易于理解的、美观的。 |
| **鼠标 / 焦点** | 作品卡 hover 可用 **边框色一阶或极低透明蒙层**（莫兰迪 secondary），时长 ≤200 ms。键盘 Tab 聚焦 **focus-visible** 清晰可见；若系统开启减弱动效，**切换与 hover 动画关闭或改为瞬时**。**原则对照**：细致的、诚实的。 |
| **加载与媒体** | **弃用老式电视机等拟物长加载**；首屏与作品图使用 **与底色协调的轻占位 + 图片淡入**。列表项分批出现若需要，仅用 **极短 opacity**，总视觉节奏保持静默。**原则对照**：诚实的、耐久的、尽可能少的设计。 |
